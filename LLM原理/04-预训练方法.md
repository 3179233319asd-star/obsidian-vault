# 四、预训练方法

## 【18】自监督学习

**是什么**：一种不需要人工标注的训练范式——模型从数据本身自动构造监督信号。在 LLM 中，做法是用文本的一部分去预测另一部分（比如用上文预测下文），标签就是文本自身。

**为什么这样设计**：人工标注极其昂贵，互联网上却有近乎无限的文本。自监督学习的精妙之处在于它把"读文本"变成了"做练习题"——给你一段话的前半段，后半段就是标准答案。模型在大量"自测"中学会了语言的统计规律——语法结构、语义关系、常识推理。它不需要知道什么是"名词"什么是"主语"，只需要反复预测下一个词，就自然而然地内化了这些语言学概念。

**和周边的联系**：自监督学习是预训练的理论基础。CLM 和 MLM 是自监督学习在 LLM 中的两种具体实现方式。预训练后仍需人工标注数据做 SFT 和对齐——自监督让你能造出通用智能，但要让这个智能遵循指令和安全规范，还是需要人类示范和反馈。

### 面试问答

**Q：自监督、无监督、有监督的区别？**

A：有监督 = 人工标签，无监督 = 没标签自己找规律，自监督 = 从数据自身构造标签（如下词预测），形式上像有监督但无需人工成本。

**Q：LLM 预训练为什么是自监督？**

A：任务"预测下一个词"的标签就是原文本的下一个 token，不需要任何人标注。互联网所有文本自动变成练习题。

**Q：能替代人工标注吗？**

A：不能完全替代。自监督学到"语言是什么样的"，但没学"人类想要什么样的回答"。指令遵循和安全边界仍需人工标注。

## 【19】因果语言建模（CLM）

**是什么**：自回归训练任务——给定序列的前 t-1 个 token，预测第 t 个 token。使用因果注意力保证每个位置只能看到左边。GPT 全系列用此方法。

**为什么这样设计**：文本生成的过程天然是自回归的——你写第 10 个词时只能基于前 9 个词做决定。CLM 在训练时模拟了完全相同的条件——每个词都只能基于它的上文来预测。这确保了训练和推理之间不存在"鸿沟"——训练时怎么做的，推理时就怎么做。反之，如果训练时模型能看双向上下文，推理时却没有"未来"可看，训练学到的能力就迁移不过去。CLM 的效率也很高——一次前向传播就能同时计算整个序列所有位置的损失。

**和周边的联系**：CLM 必须搭配 Decoder-only 架构和因果注意力才能运作。它是预训练阶段唯一的目标函数。也是后续 SFT 阶段使用的同一种训练格式（只是换了数据）。

### 面试问答

**Q：CLM 和 MLM 的核心区别？**

A：CLM = 续写，用上文预测下一个词，单向。MLM = 完形填空，遮住中间词用左右上下文猜，双向。GPT 用 CLM，BERT 用 MLM。

**Q：为什么生成式 LLM 用 CLM？**

A：训练和推理要一致。生成时只能基于已写内容决定下一个词，CLM 训练时就是这么做。MLM 训练时看双向，推理时没双向可看，有鸿沟。

**Q：CLM 能利用双向信息吗？**

A：不能，只能用左侧。但这不是缺陷——真实生成时也只有左侧可用。训练时老老实实模拟真实场景。

## 【20】掩码语言建模（MLM）

**是什么**：BERT 的核心训练任务——随机遮住输入中 15% 的 token（大部分用 [MASK] 替换，小部分替换为随机词或保留原词），模型基于未被遮住的上下文（双向）预测被遮住的原始 token。

**为什么这样设计**：MLM 让模型被迫学会"根据周围的环境推断缺失的信息"。因为预测一个被遮住的词时，你必须同时利用它左边和右边的上下文——「我喝了 [MASK] 牛奶」，只有左右一起看才能锁定是"一杯"类的词而非其他。这迫使模型把双向信息都编码进隐状态，从而获得更强的语言理解能力。15% 的比例是经过精心设计的——太高会让 [MASK] 和真实输入的分布差异太大，太低则训练信号不够。实际还做了 80/10/10 的替换策略，让模型不完全依赖 [MASK] 符号。

**和周边的联系**：MLM 必须搭配 Encoder（双向注意力）才能工作。它不适合生成任务——训练时有 [MASK] 和双向上下文，生成时两者都不存在。BERT 系列是 MLM 的代表。

### 面试问答

**Q：BERT 为什么只遮 15% 不是全部？**

A：全遮 → 训练时只见过 [MASK]，微调时没有 [MASK]，输入分布不匹配。15% 是平衡点——有信号又不过度依赖 [MASK]。

**Q：MLM 怎么学到双向上下文？**

A：预测被遮住的词时必须综合左右两侧信息，光看左边猜不准，光看右边也猜不准。双向信息被强制融合。

**Q：为什么 GPT 不用 MLM？**

A：GPT 做生成需要自回归。MLM 依赖 [MASK] 和双向上下文，生成时两个都不存在，训练和推理不匹配。
