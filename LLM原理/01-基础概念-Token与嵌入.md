# 一、基础概念（输入表示）

## 【1】Token / 分词

**是什么**：把原始文本切成一个个"子词"单元，每个单元对应词表里的一个编号。比如「我喜欢吃苹果」可能被切成「我」「喜欢」「吃」「苹果」四个 token。

**为什么这样设计**：自然语言的词汇是无限的（每天都有新词出现），但模型只能处理有限的输入。分词算法（BPE、SentencePiece）的做法是把高频常用词作为一个整体保留（比如「the」「喜欢」），把低频罕见词拆成更小的子词片段（比如「antidisestablishmentarianism」→「anti」「dis」「establish」「ment」「arian」「ism」）。这样无论遇到什么新词，都能用已有子词拼出来。可以理解为：模型手里有一盒乐高积木（子词），每次遇到新词语就用积木拼出来。

**和周边的联系**：分词的输出是 token ID 序列，下一步会被送入嵌入层变成向量。分词决定了序列长度——序列越长，后续注意力计算越贵（O(L²)）。

### 面试问答

**Q：为什么不直接按字符处理？**

A：字符级序列太长，注意力计算量按长度平方涨，扛不住；而且单个字符没有语义，模型很难学习。分词兼顾了长度和语义。

**Q：BPE 和 SentencePiece 的区别？**

A：BPE 依赖空格识别词边界，中文不友好。SentencePiece 把空格当普通字符，不假定语言结构，多语言适用。LLaMA 等现代模型都用 SentencePiece。

**Q：词表大小一般多少？**

A：32k~100k。太小序列变长、训练慢；太大嵌入层参数爆炸、稀有词学不好。32k~100k 是实践中性价比最高的区间。

## 【2】嵌入（Embedding）

**是什么**：一个巨大的查找表（矩阵），每一行是一个 token 对应的稠密向量。比如 token ID = 1532 对应第 1532 行的向量。向量的维度通常是 768~12288，每个维度编码了这个词在某方面（语义、语法、用法）的特征。

**为什么这样设计**：神经网络只能做连续向量的计算，不能处理离散的编号。嵌入层就是"离散 → 连续"的桥梁。更妙的是，经过训练后，语义相近的词会自动被映射到向量空间中相近的位置——「猫」和「狗」的向量夹角很小，「猫」和「汽车」的夹角很大。这意味着模型可以通过简单的向量运算捕捉语义关系，比如「国王 - 男人 + 女人 ≈ 女王」。

**和周边的联系**：嵌入矩阵是整个模型中参数量最大的部分之一（词表大小 × 嵌入维度），也是模型第一个被训练的组件。嵌入向量做出来后加上位置编码，就是喂给 Transformer 的实际输入。

### 面试问答

**Q：嵌入维度怎么确定？**

A：跟模型规模挂钩。太小装不下足够语义信息，太大参数和显存暴涨。小模型 768，中模型 1024~4096，大模型 4096~12288。

**Q：预训练后嵌入层还能微调吗？**

A：能，通常跟着一起更新。但下游数据少时冻结嵌入层更安全，避免破坏预训练学到的通用语义。

**Q：为什么语义相近的词向量也相近？**

A：训练目标是预测下一个词。「猫」和「狗」出现在几乎相同的上下文里，模型为了在这些位置都能预测对，自然把它们推到了相近的位置。

## 【3】位置编码

**是什么**：给每个 token 的嵌入向量叠加上一个和位置相关的信号。有了位置编码，模型才知道「我爱你」的第一个词是「我」、第三个词是「你」——而不只是"这三个词存在"。

**为什么这样设计**：Transformer 的注意力机制本质是"计算每两个 token 之间的相关度"，它只看关系不看顺序。把句子所有词放进一个袋子摇晃，注意力算出来的结果一模一样。所以必须额外注入位置信息，让模型知道谁在前谁在后。RoPE（旋转位置编码）是当前主流方案，它不把位置编码直接加到向量上，而是把 Q 和 K 向量各自旋转一个跟位置成正比的角度——这样 Q_i 和 K_j 的点积结果只取决于相对距离 (i-j)，天然支持比训练时更长的序列。

**和周边的联系**：位置编码直接影响了 Transformer 注意力层的计算（Q 和 K 的旋转→点积），也是长上下文外推技术的基础——调整位置编码的缩放方式就可以让模型处理训练时没见过的长度。

### 面试问答

**Q：为什么需要位置编码？**

A：注意力只看关系不看顺序，不注入位置信息就分不清"我爱你"和"你爱我"。位置编码给每个 token 打上"第几个"的标签。

**Q：绝对位置编码和 RoPE 的区别？**

A：绝对位置给每个位置固定编号，简单但换更长序列就失效。RoPE 编码相对距离，天然支持序列外推。目前主流用 RoPE。

**Q：RoPE 怎么做到相对位置的？**

A：把 Q 和 K 向量旋转跟位置成正比的角度。点积时角度差 = 相对距离，结果只跟位置差有关，跟绝对位置无关。
