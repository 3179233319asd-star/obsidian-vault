# 函数

函数 = 把一段可复用的代码**打包**起来，起个名字，需要时调用。

生活类比：你把"泡咖啡"的步骤写成一张卡片（函数定义），每次想喝只需喊一声"泡咖啡！"（函数调用），不用每次重新想步骤。

---

## 一、定义和调用

```python
# 定义函数
def greet():
    """向世界问好"""       # 文档字符串（docstring），写给人类看的
    print("Hello, World!")

# 调用函数
greet()     # 输出：Hello, World!
greet()     # 可以反复调用
greet()

# ⚠️ 常见错误：忘记括号！
# greet      # 不会执行！只是引用了函数对象本身
```

**关键要点：**
- `def` 关键字：告诉 Python "我要定义一个函数了"
- 函数名后必须有括号 `()`
- 函数体要缩进 4 格
- 函数**不调用就不执行**

## 二、参数

### 2.1 位置参数（最基础的方式）

按位置一一对应传参：

```python
def introduce(name, age, city):
    print(f"我叫{name}，{age}岁，住在{city}。")

introduce("小明", 25, "北京")    # 按顺序对应
# ⚠️ 数量必须完全匹配，多了少了都报错
```

### 2.2 关键字参数（不记顺序）

调用时用 `参数名=值` 的形式，顺序随意：

```python
introduce(city="广州", name="小李", age=22)  # 打乱顺序也没问题
introduce("小王", city="杭州", age=26)       # 和位置参数混用
# ⚠️ 规则：位置参数必须在关键字参数前面！
# introduce(name="小王", 26, "杭州")  # ❌ SyntaxError
```

> Java/JS 没有原生关键字参数，通常用对象 `{name: "xx", age: 20}` 模拟。

### 2.3 默认值参数

```python
def create_user(name, age, role="普通用户", active=True):
    status = "激活" if active else "未激活"
    print(f"{name}，{age}岁，角色：{role}，状态：{status}")

create_user("小明", 25)                            # 用默认值
create_user("管理员", 30, role="admin")             # 覆盖 role
create_user("测试", 20, active=False)               # 覆盖 active
```

> **规则：有默认值的参数必须放在无默认值参数的后面。**

### ⚠️ 默认值陷阱（重要！）

**永远不要用可变对象（list/dict/set）作为默认参数值！**

```python
# ❌ 危险写法：默认列表被所有调用共享！
def add_item_bad(item, items=[]):
    items.append(item)
    return items

add_item_bad("苹果")   # ['苹果'] — 看起来正常
add_item_bad("香蕉")   # ['苹果', '香蕉'] — 💥 "苹果"怎么还在？

# 原因：默认值 [] 只在函数定义时创建一次，
# 后续调用都共享同一个列表对象！

# ✅ 正确写法：用 None 代替
def add_item_good(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

### 2.4 可变参数 *args

把多余的**位置参数**打包进一个**元组**：

```python
def sum_all(*nums):
    print(f"收到了 {len(nums)} 个数：{nums}")
    return sum(nums)

sum_all(1, 2)           # 2 个数，返回 3
sum_all(1, 2, 3, 4, 5)  # 5 个数，返回 15
sum_all()               # 0 个数，返回 0

# 解包调用：* 可以把列表/元组拆成位置参数
def add3(a, b, c):
    return a + b + c

nums = [10, 20, 30]
add3(*nums)    # 等价于 add3(10, 20, 30)
```

### 2.5 可变关键字参数 **kwargs

把多余的**关键字参数**打包进一个**字典**：

```python
def print_info(**info):
    for key, value in info.items():
        print(f"  {key}: {value}")

print_info(name="张三", age=25, city="北京")

# 解包调用：** 可以把字典拆成关键字参数
person = {"name": "小明", "age": 25, "city": "北京"}
introduce(**person)   # 等价于 introduce(name="小明", age=25, city="北京")
```

### 2.6 强制关键字参数（`*`）

`*` 后面的参数**必须用关键字传递**，防止调用者搞混：

```python
def connect(host, port, *, timeout=30, use_ssl=True):
    print(f"连接 {host}:{port}，超时={timeout}s，SSL={use_ssl}")

connect("localhost", 8080, timeout=60)
# connect("localhost", 8080, 60)  # ❌ TypeError: 必须用关键字
```

### 2.7 仅限位置参数（`/`，Python 3.8+）

`/` 前面的参数**只能用位置传递**，不能用关键字：

```python
def power(base, exp, /):
    return base ** exp

power(2, 10)       # ✅
# power(base=2, exp=10)  # ❌ TypeError
```

### 参数顺序总结

```python
def func(仅位置参数, /, 普通参数, 有默认值参数, *, 仅关键字参数):
    pass
```

## 三、返回值 return

### 基本用法

```python
def add(a, b):
    return a + b          # return = 把结果交还给调用者

result = add(3, 5)        # 8
print(add(2, 3))          # 5 — 可以直接在表达式中用
```

**没有 return → 返回 None。**

### 提前返回（Early Return）

```python
def check_age(age):
    if age < 0:
        return "无效年龄"     # 提前返回
    if age < 18:
        return "未成年"
    return "成年"
```

### 多返回值（Python 杀手级特性）

```python
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder   # 逗号分隔 → 自动打包成元组

q, r = divide(17, 5)     # 元组解包接收
# q=3, r=2

result = divide(17, 5)   # (3, 2) — 也可以接整个元组

# Python 内置的 divmod 就是典型例子
q, r = divmod(17, 5)
```

**用 `_` 忽略不需要的返回值：**

```python
_, _, avg = stats([1, 2, 3, 4, 5])     # 只要平均值
first, *_, last = [1, 2, 3, 4, 5, 6]   # 只要首尾
```

### 返回值太多时：用字典或命名元组

```python
# ✅ 方案1：返回字典（调用方通过键名访问）
def analyze_text(text):
    words = text.split()
    return {
        "char_count": len(text),
        "word_count": len(words),
        "unique_words": len(set(words))
    }

info = analyze_text("hello world hello")
print(info["word_count"])

# ✅ 方案2：命名元组（像轻量级 class）
from collections import namedtuple

def get_point():
    Point = namedtuple("Point", ["x", "y", "z"])
    return Point(0, 0, 0)

origin = get_point()
print(f"x={origin.x}, y={origin.y}")

# ✅ 方案3：dataclass（Python 3.7+，后面章节详讲）
from dataclasses import dataclass

@dataclass
class UserInfo:
    name: str
    age: int
    email: str
```

## 四、变量作用域（LEGB 规则）

Python 按这个顺序查找变量：**L**ocal → **E**nclosing → **G**lobal → **B**uilt-in

```python
x = "全局的 x"                    # G - Global

def outer():
    x = "外层函数的 x"            # E - Enclosing
    def inner():
        x = "内层函数的 x"        # L - Local
        print(x)                  # 打印"内层函数的 x"
    inner()
    print(x)                      # 打印"外层函数的 x"

outer()
print(x)                          # 打印"全局的 x"
```

### ⚠️ Python 没有块级作用域！

Java/JS 的 `{}` 内变量出不来，但 Python 中 if/for **不创建新作用域**：

```python
if True:
    block_var = "我在 if 块里创建的"
print(block_var)     # ✅ 正常访问！Python 不限制

for i in range(5):
    loop_var = i
print(i, loop_var)   # ✅ 正常访问！i=4, loop_var=4
```

Python 中只有**函数、类、模块**会创建新的作用域。

### 赋值即定义（关键规则）

在函数内对变量赋值，Python 就认为它是**局部变量**：

```python
counter = 100  # 全局变量

def broken():
    # ❌ UnboundLocalError！
    # Python 看到 counter = ... 认为 counter 是局部变量
    # 但赋值前就用了它 → 报错
    counter = counter + 1
```

### global —— 修改全局变量

```python
score = 0

def add_score(points):
    global score           # 声明：我要修改全局的 score
    score += points

add_score(10)  # score = 10
add_score(20)  # score = 30
```

> **尽量少用 global，推荐通过参数传入、返回值传出。**

### nonlocal —— 修改外层函数变量

```python
def make_counter(start=0):
    count = start
    def increment():
        nonlocal count          # 声明修改外层的 count
        count += 1
        return count
    return increment

counter = make_counter()
print(counter())   # 1
print(counter())   # 2
print(counter())   # 3  ← 每次调用都累加
```

### 闭包（Closure）

闭包 = 内层函数 + 它引用的外层变量。即使外层函数执行完毕，内层仍能访问外层变量：

```python
def make_multiplier(factor):
    """创建一个乘以固定倍数的函数"""
    def multiply(x):
        return x * factor    # factor 来自闭包，只读不需要 nonlocal
    return multiply

double = make_multiplier(2)
triple = make_multiplier(3)

double(5)   # 10
triple(5)   # 15
```

### ⚠️ 循环中的闭包陷阱

```python
# ❌ 有问题：所有函数引用同一个 i
functions = []
for i in range(5):
    functions.append(lambda: i)    # i 是闭包引用

print([f() for f in functions])    # [4, 4, 4, 4, 4] ← 全是 4！

# ✅ 修复1：用默认参数"捕获"当前值
functions = []
for i in range(5):
    functions.append(lambda x=i: x)  # x=i 在定义时就确定了

print([f() for f in functions])      # [0, 1, 2, 3, 4] ✅

# ✅ 修复2：用工厂函数
def make_func(val):
    return lambda: val
functions = [make_func(i) for i in range(5)]
```

## 五、lambda —— 匿名函数

只能写**一行表达式**，自动 return。不需要起名字。

```python
# 传统函数
def add(a, b):
    return a + b

# lambda 写法
add = lambda a, b: a + b
add(3, 5)   # 8

# ⚠️ 不推荐把 lambda 赋值给变量！需要名字直接用 def
```

### lambda 最常用的场景

```python
# 1. 排序的 key
students.sort(key=lambda s: s["score"], reverse=True)
words = sorted(words, key=lambda w: len(w))

# 2. 多条件排序（元组天然支持多级比较）
sorted(items, key=lambda x: (x["type"], x["price"]))

# 3. 数字降序用取负号
sorted(students, key=lambda s: (s["class"], -s["score"]))

# 4. 条件表达式可以放 lambda 里
check = lambda x: "正数" if x > 0 else ("零" if x == 0 else "负数")

# ❌ 逻辑复杂就用 def，不要强行 lambda
```

## 六、高阶函数

函数可以作为参数传给另一个函数。

### map —— 对每个元素执行操作

```python
nums = [1, 2, 3, 4, 5]
list(map(str, nums))                     # ["1", "2", "3", "4", "5"]
list(map(lambda x: x ** 2, nums))        # [1, 4, 9, 16, 25]

# map 可以处理多个可迭代对象
a, b = [1, 2, 3], [10, 20, 30]
list(map(lambda x, y: x + y, a, b))      # [11, 22, 33]

# 实用：字符串列表转整数列表
list(map(int, ["10", "20", "30"]))        # [10, 20, 30]
```

### filter —— 按条件筛选

```python
nums = [-2, -1, 0, 1, 2, 3]
list(filter(lambda x: x > 0, nums))       # [1, 2, 3]

# filter(None) → 过滤掉所有假值（0, "", None 等）
list(filter(None, [0, 1, "", "hello", None]))
# [1, "hello"]
```

### reduce —— 累积/折叠（需导入）

```python
from functools import reduce

reduce(lambda a, b: a + b, [1, 2, 3, 4])    # 10
reduce(lambda a, b: a * b, [1, 2, 3, 4])    # 24

# 带初始值
reduce(lambda a, b: a + b, [1, 2, 3, 4], 100)   # 110

# 很多 reduce 场景有更简单的替代：
sum([1, 2, 3, 4])           # 替代 reduce(+, ...)
math.prod([1, 2, 3, 4])     # 替代 reduce(*, ...)
" ".join(words)             # 替代 reduce(拼接)
```

> map/filter 返回**惰性迭代器**，需要 `list()` 转换才能看到结果。大多数情况下列表推导式比 map/filter 更 Pythonic。

### operator 模块（替代简单 lambda）

```python
from operator import itemgetter, attrgetter

# itemgetter 替代 lambda x: x[key]
sorted(products, key=itemgetter("price"))

# 多字段排序
sorted(students, key=itemgetter("grade", "score"))
```

## 七、递归 —— 函数调用自己

### 两要素

1. **基准条件（Base Case）** — 什么时候停下来
2. **递归步骤（Recursive Step）** — 把问题缩小，调用自身

```python
def factorial(n):
    if n <= 1:                       # 基准条件（必须有！）
        return 1
    return n * factorial(n - 1)      # 递归步骤

factorial(5)   # 5 × 4 × 3 × 2 × 1 = 120
```

执行过程：
```
factorial(5) = 5 * factorial(4)
             = 5 * 4 * factorial(3)
             = 5 * 4 * 3 * factorial(2)
             = 5 * 4 * 3 * 2 * factorial(1)
             = 5 * 4 * 3 * 2 * 1 = 120
```

### 记忆化优化（解决重复计算）

```python
# ❌ fib(30) 需要上百万次函数调用，因为有大量重复计算
def fib(n):
    if n <= 0: return 0
    if n == 1: return 1
    return fib(n-1) + fib(n-2)

# ✅ 方法1：用 functools.lru_cache（最简洁）
from functools import lru_cache

@lru_cache(maxsize=None)
def fib_cached(n):
    if n <= 0: return 0
    if n == 1: return 1
    return fib_cached(n-1) + fib_cached(n-2)

fib_cached(100)   # 瞬间算出！

# ✅ 方法2：手动字典缓存
def fib_memo(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 0: return 0
    if n == 1: return 1
    memo[n] = fib_memo(n-1, memo) + fib_memo(n-2, memo)
    return memo[n]
```

### Python 递归深度限制

```python
import sys
print(sys.getrecursionlimit())   # 默认 1000

# 超出限制 → RecursionError: maximum recursion depth exceeded
```

### 递归擅长什么？

| ✅ 适合 | ❌ 不适合 |
|---------|----------|
| 树形/嵌套结构遍历 | 简单循环就能搞定的 |
| 分治算法（快排、归并） | 递归深度可能很大的（>1000） |
| 数学递推公式 | 性能敏感场景 |
| 回溯算法（排列组合） | 能用内置函数解决的问题 |

### 经典递归算法

```python
# 二分查找（递归版）
def binary_search(arr, target, low=0, high=None):
    if high is None:
        high = len(arr) - 1
    if low > high:
        return -1
    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search(arr, target, mid + 1, high)
    else:
        return binary_search(arr, target, low, mid - 1)

# 快速排序
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    less = [x for x in arr[1:] if x <= pivot]
    greater = [x for x in arr[1:] if x > pivot]
    return quicksort(less) + [pivot] + quicksort(greater)

# 扁平化嵌套列表
def flatten(lst):
    result = []
    for item in lst:
        if isinstance(item, list):
            result.extend(flatten(item))
        else:
            result.append(item)
    return result
```

> Python **不做尾递归优化**，性能关键场景改为循环。能用内置函数（sum、max、math.prod）就别自己写递归。

## 八、实战：常用函数写一遍

```python
# 1. 判断质数（for-else 妙用）
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    else:
        return True

# 2. 统计字符串中每个字符出现次数
def char_count(s):
    result = {}
    for ch in s:
        result[ch] = result.get(ch, 0) + 1
    return result

# 3. 安全除法（仿 Go 风格）
def safe_divide(a, b):
    if b == 0:
        return None, False, "除数不能为0"
    return a / b, True, ""

result, ok, err = safe_divide(10, 0)
if not ok:
    print(f"错误：{err}")
```

## 新手最容易踩的坑

### 坑1：默认参数用可变对象

```python
# ❌ 默认列表只创建一次，被所有调用共享
def add_item(item, lst=[]):
    lst.append(item)
    return lst
# ✅ 用 None 代替
def add_item(item, lst=None):
    if lst is None: lst = []
    lst.append(item)
    return lst
```

### 坑2：忘记调用函数（没加括号）

```python
def greet(): print("Hello")
greet      # 不执行！只是引用了函数对象
greet()    # ✅ 有括号才执行
```

### 坑3：全局变量忘记 global

```python
count = 0
def increment():
    global count
    count += 1    # ✅ 不加 global 会 UnboundLocalError
```

```python
# ❌ 所有函数都返回同一个值（循环结束后的值）
# ✅ 用默认参数捕获：lambda x=i: x
```

### 坑6：return 位置不对

```python
# ❌ 只检查了第一个元素就返回了
for n in nums:
```
