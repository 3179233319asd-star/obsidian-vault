# 运算符

运算符就是用来**做运算**的符号。就像数学课上的 `+`、`-`、`×`、`÷`，Python 也有自己的一套。

## 一、算术运算符

```python
a = 17
b = 5

a + b    # 22 — 加
a - b    # 12 — 减
a * b    # 85 — 乘
a / b    # 3.4 — 除（注意：永远返回浮点数）
a // b   # 3  — 整除（只保留整数部分）
a % b    # 2  — 取余（除不完剩下的部分）
a ** b   # 17⁵ — 幂运算（a 的 b 次方）
```

### `/` 和 `//` 的区别（重点！）

这是初学者最容易搞混的地方。

```python
7 / 2     # 3.5 → 普通除法，有小数
7 // 2    # 3   → 整除，"7 里面最多能装几个 2"

# // 是向下取整
-7 // 3   # -3（不是 -2！因为 -2.333... 向下取整是 -3）
```

### `//` 和 `%` 是一对

它们经常一起出现，解决同一个问题的两个方面：

```
17 ÷ 5 = 3 ... 余 2
17 // 5 = 3    （商）
17 % 5 = 2     （余数）
```

实用场景——把秒数改成时间：

```python
total = 3725

hours = total // 3600            # 1 小时
minutes = (total % 3600) // 60   # 2 分钟
seconds = total % 60             # 5 秒

print(f"{total}秒 = {hours}小时{minutes}分钟{seconds}秒")
```

判断奇偶数：

```python
num = 7
if num % 2 == 0:
    print("偶数")
else:
    print("奇数")
```

### 乘方 `**`

```python
2 ** 10    # 1024 — 2 的 10 次方
9 ** 0.5   # 3.0  — 9 的平方根（就是开根号）
```

### 运算优先级

优先级和数学课一样：

```
**（幂运算）> 正负号 > * / // % > + -
```

```python
2 + 3 * 4     # 14 — 先乘后加
(2 + 3) * 4   # 20 — 括号优先
2 ** 3 ** 2   # 512 — 幂运算是从右往左算的
(2 ** 3) ** 2 # 64
```

如果记不住——**加括号就对了**。

## 二、比较运算符

比较的结果是 `True` 或 `False`。

```python
x = 10
y = 20

x == y   # False — 等于？不是（注意是两个等号！）
x != y   # True  — 不等于？对
x > y    # False — 大于？不是
x < y    # True  — 小于？对
x >= y   # False — 大于等于？不是
x <= y   # True  — 小于等于？对
```

### `=` 和 `==` 完全不同！

```python
score = 100        # 一个等号 = 赋值：把 100 放进 score
score == 100       # 两个等号 == 比较：score 等于 100 吗？返回 True
```

**记住：`=` 是"把...放进去"，`==` 是"等于吗？"。**

### 链式比较（Python 独有！）

在其他语言里要写 `x > 10 and x < 20`，Python 可以更直观：

```python
age = 25
18 <= age <= 60    # True — age 在 18 到 60 之间吗？

score = 85
80 <= score < 90   # True — score 在 80 到 90 之间吗？

# 等价于
score >= 80 and score < 90
```

### 字符串也能比较

```python
"apple" == "apple"    # True
"apple" < "banana"    # True — 按字典顺序
# 原理：比较字符的编码值（a=97, b=98, ... A=65, B=66）
# 大写字母 < 小写字母
```

## 三、逻辑运算符

用来**组合多个条件**。

```python
age = 25
has_id = True

# and → 两个条件都要满足
age >= 18 and has_id    # True — 年龄够 且 有身份证

# or → 满足一个就行
is_vip = False
is_employee = True
is_vip or is_employee   # True — 是 VIP 或是员工

# not → 反过来
is_raining = False
not is_raining          # True — 没下雨
```

### 短路求值——省事的计算

`and` 和 `or` 不会傻算到底，一旦能确定结果就停了：

```python
# and：左边为 False → 右边不看了，总体肯定是 False
False and print("不会执行")   # print 不执行！

# or：左边为 True → 右边不看了，总体肯定是 True
True or print("不会执行")     # print 不执行！
```

### Python 的逻辑运算符返回的是操作数本身

```python
1 and 2      # 2 — 1 是 True，看第二个，返回 2
0 and 2      # 0 — 0 是 False，直接返回 0
1 or 2       # 1 — 1 是 True，直接返回 1
0 or 2       # 2 — 0 是 False，返回第二个

# 实用技巧：给变量设默认值
username = ""
display = username or "匿名用户"
print(display)   # "匿名用户"
```

这段代码的意思是：如果 `username` 是空字符串（False），就用 `"匿名用户"` 代替。这是个很常见的 Python 写法。

### 优先级

`not` > `and` > `or`。不确定就加括号。

## 四、赋值运算符

```python
x = 10        # 基本赋值
x += 5        # x = x + 5 → 15
x -= 3        # x = x - 3 → 12
x *= 2        # x = x * 2 → 24
x /= 4        # 6.0
x //= 5       # x = x // 5
x %= 10       # x = x % 10
x **= 3       # x = x ** 3
```

### Python 没有 `++` 和 `--`！

Java、JavaScript、C 里面可以写 `i++`，**Python 不行**。想写 `i = i + 1` 的简写，就是 `i += 1`。

## 五、成员运算符 `in` / `not in`

判断某个东西是否在某个"容器"里。

```python
# 在字符串里找
"a" in "abc"                  # True
"Python" in "我爱学Python"     # True
"Java" in "我爱学Python"       # False

# 在列表里找
fruits = ["苹果", "香蕉", "橘子"]
"香蕉" in fruits               # True
"西瓜" in fruits               # False
"西瓜" not in fruits           # True

# 在字典里找（查的是键）
"name" in {"name": "小明"}     # True
```

## 六、身份运算符 `is` / `is not`

```python
# == 比较两个变量的"值"是否相等（长得一样吗？）
# is 比较两个变量是不是"同一个东西"（是同一个人吗？）
```

```python
a = [1, 2, 3]
b = [1, 2, 3]    # b 是另一个列表，虽然内容一样
c = a             # c 和 a 指向同一个列表

a == b    # True — 内容相同
a is b    # False — 但不是同一个对象
a is c    # True — c 就是 a
```

用生活中的例子理解：
- `==` 问的是"你俩长得像不像？"
- `is` 问的是"你俩是不是同一个人？"

**判断 `None` 必须用 `is`：**

```python
result = None
if result is None:         # ✅ 推荐
    print("没有结果")
if result is not None:     # ✅ 推荐
    print("有结果")
# result == None           # ❌ 不推荐
```

## 七、位运算符（了解即可）

对数字的二进制位进行操作。日常编程少用，但面试可能问到。

```python
a = 0b1010   # 二进制的 10
b = 0b1100   # 二进制的 12

a & b    # 8  — 按位与
a | b    # 14 — 按位或
a ^ b    # 6  — 按位异或
~a       # -11 — 按位取反
a << 1   # 20 — 左移一位（相当于乘 2）
a >> 1   # 5  — 右移一位（相当于除 2）
```

## 八、运算符优先级总表

从高到低：

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 🥇 | `()` | 括号最优先 |
| 🥈 | `**` | 幂运算 |
| 🥉 | `+x` `-x` `~x` | 正负号、按位取反 |
| 4 | `*` `/` `//` `%` | 乘除整除取余 |
| 5 | `+` `-` | 加减 |
| 6 | `<<` `>>` | 位移 |
| 7 | `&` | 按位与 |
| 8 | `^` | 按位异或 |
| 9 | `\|` | 按位或 |
| 10 | `==` `!=` `<` `>` `<=` `>=` `is` `in` | 比较和判断 |
| 11 | `not` | 逻辑非 |
| 12 | `and` | 逻辑与 |
| 13 | `or` | 逻辑或 |

> 记不住没关系，**不确定的时候就加括号**。

## 九、新手最容易踩的坑

### 坑1：`=` 和 `==` 搞混

这是新手犯最多的错误！在所有编程语言里都是。

```python
# ❌ if 条件里写成了赋值
if age = 18:         # 报错！= 是赋值，不能放在 if 里判断
    print("成年")

# ✅ 比较必须用 ==
if age == 18:
    print("成年")
```

### 坑2：把 Python 的逻辑运算符写成符号

从 Java / JS 转过来的同学最容易踩。

```python
# ❌ Python 不认这些
# if age > 18 && has_id:      # && 是 Java/JS 的写法
# if is_vip || is_admin:      # || 是 Java/JS 的写法
# if !is_banned:              # ! 是 Java/JS 的写法

# ✅ Python 必须用英文单词
if age > 18 and has_id:
if is_vip or is_admin:
if not is_banned:
```

### 坑3：Python 没有 `++` 和 `--`

```python
# ❌ Python 不认
# i++
# ++i
# i--

# ✅ 只能写
i += 1
i -= 1
```

### 坑4：`/` 和 `//` 搞混

```python
# Java/JS 里整数除以整数 = 整除
# Python 里 / 永远返回小数！
7 / 2      # 3.5 — 普通除法
7 // 2     # 3   — 这才是整除

# 如果要整除，记得用 //
```

### 坑5：`is` 和 `==` 分不清

```python
a = [1, 2, 3]
b = [1, 2, 3]

a == b     # True — 值一样
a is b     # False — 但不是同一个对象

# is 的主要用途：判断 None
if x is None:       # ✅
if x is not None:   # ✅
# if x == None:     # ❌ 不推荐
```

### 坑6：and/or 返回的不一定是 True/False

```python
# Python 的 and/or 返回的是操作数本身
result = "" or "默认值"
print(result)    # "默认值" — 不是 True！

# 这在设默认值时很有用，但别搞混了
```

### 坑7：比较不同类型会出意外

```python
# ❌ Python 3 中不同类型不能直接比大小
# "42" > 10         # TypeError

# ✅ 先转成同一类型
int("42") > 10      # True
```
