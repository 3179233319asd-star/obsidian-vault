# 变量与数据类型

## 一、print —— 让电脑说话

学编程的第一步，就是让电脑把信息显示在屏幕上。`print()` 就是干这个的。

你可以把 `print()` 想象成一个**喇叭**——把想说的话放进喇叭里，电脑就会喊出来。

```python
print("你好，世界！")
print("Hello, World!")
```

### 引号

文字用引号包起来。单引号、双引号都可以，效果一样。

```python
print('用单引号也行')
print("用双引号也行")
```

问：什么时候用哪种？答：当文字里面有引号时，外面用另一种。

```python
print("他说：'你好'")    # 文字里有单引号，外面用双引号
print('他说："你好"')    # 文字里有双引号，外面用单引号
```

### 打印数字

数字不需要加引号，直接放进去。

```python
print(100)
print(3.14)
```

### 一次性打印多个东西

用**逗号**隔开。Python 会自动在中间加个空格。

```python
print("我的名字是", "小明")
print("我今年", 18, "岁")
print("1 + 1 =", 2)
```

输出效果：
```
我的名字是 小明
我今年 18 岁
1 + 1 = 2
```

### 打印空行

什么都不放，就输出一行空白。

```python
print()
```

### print 会自动换行

每次 `print()` 结束后，光标会自动跳到下一行。

```python
print("第一行")
print("第二行")
```

输出：
```
第一行
第二行
```

如果你不想换行怎么办？用 `end` 参数：

```python
print("加载中", end="")
print(".", end="")
print(".", end="")
print(".")
# 输出：加载中...
```

`end` 的意思是"打印完这个后，结尾用什么"，默认是 `\n`（换行符）。你把它改成空字符串 `""`，就不会换行了。

更实用的例子：

```python
print("苹果", end="、")
print("香蕉", end="、")
print("橘子")
# 输出：苹果、香蕉、橘子
```

### 自定义分隔符

```python
print("苹果", "香蕉", "橘子", sep="、")
# 输出：苹果、香蕉、橘子

print(2026, 5, 9, sep="-")
# 输出：2026-5-9
```

`sep` 的意思是"多个值之间用什么隔开"，默认是空格。

## 三、变量 —— 贴了标签的盒子

### 生活中的比喻

想象你面前有很多**盒子**：
- 每个盒子上面贴了一个**标签**（名字）
- 盒子里面可以放**东西**（值）

比如：
- 有一个盒子，标签写着 `age`，里面放着数字 `18`
- 有一个盒子，标签写着 `name`，里面放着文字 `"小明"`

在 Python 里，这个"贴了标签的盒子"就叫做**变量**。

### 创建变量

```python
age = 18
name = "小明"
height = 1.75

print("姓名：", name)
print("年龄：", age)
print("身高：", height)
```

`=` 不是数学里的"等于"，而是**赋值**——把右边的值放进左边名字的盒子里。

### 变量可以随时换里面的值

```python
score = 60
print("期中成绩：", score)   # 60

score = 90
print("期末成绩：", score)   # 90
```

盒子还是那个盒子（`score`），但里面的东西从 60 换成了 90。

### 变量可以存不同类型的数据

```python
a = 100        # 放一个整数
print(a)       # 100

a = "一百"     # 换成一段文字
print(a)       # 一百

a = True       # 换成一个判断结果
print(a)       # True
```

同一个变量，可以先放数字，后放文字，再放对错判断——Python 不在乎。

### 同时创建多个变量

```python
x, y, z = 1, 2, 3     # 三个变量同时赋值
print(x, y, z)         # 1 2 3

a = b = c = 0          # 三个变量都等于 0
```

### 交换两个变量的值（Python 独门绝技）

```python
left = "左手拿着苹果"
right = "右手拿着香蕉"

left, right = right, left

print(left)   # 左手拿着香蕉
print(right)  # 右手拿着苹果
```

在其他语言里你需要一个临时变量来交换，Python 一行搞定。

## 四、变量命名规则

变量名不是随便起的，要遵守几条规则。

### 规则一：只能包含字母、数字、下划线

```python
my_name = "小明"     # ✅ 合法
age2 = 20            # ✅ 合法
_secret = "隐藏数据"  # ✅ 合法（下划线开头也可以）

# my-name = "小明"   # ❌ 不能用减号
# my name = "小明"   # ❌ 不能有空格
# 2age = 20          # ❌ 不能以数字开头
```

### 规则二：不能以数字开头

```python
student1 = "张三"    # ✅ 数字在中间或末尾没问题
# 1student = "张三"  # ❌ 数字开头不行
```

### 规则三：不能用 Python 的关键字

关键字是 Python 自己要用到的单词，你不能抢来当变量名。

常见的：`if`、`else`、`for`、`while`、`True`、`False`、`None`、`and`、`or`、`not`、`def`、`class`、`return`、`import`、`from`……

```python
import keyword
print(keyword.kwlist)  # 查看所有关键字
```

### 规则四：区分大小写

```python
Name = "小明"
name = "小红"
NAME = "小刚"

print(Name)  # 小明
print(name)  # 小红
print(NAME)  # 小刚
# 这三个是完全不同的变量！
```

### 推荐的命名风格

Python 社区推荐使用**蛇形命名法**（snake_case）：
- 所有字母小写
- 单词之间用下划线连接

```python
student_name = "张三"    # ✅ 推荐
max_score = 100          # ✅ 推荐
is_logged_in = True      # ✅ 推荐

# 不推荐的写法（语法没错，但不合群）：
studentName = "张三"     # ⚠️ 驼峰命名，Java/JS 习惯，Python 社区不推荐
StudentName = "张三"     # ⚠️ 大驼峰，一般只有类名才这样写
```

### 起名的好习惯

变量名要**见名知意**——看名字就知道里面放的什么。

```python
# ✅ 好名字
user_age = 25
max_retry_count = 3
is_vip = True

# ❌ 坏名字
a = 25          # a 是什么？
x = 3           # x 表示什么？
flag = True     # 什么标记？
```

## 五、数字类型

### 整数 int

没有小数点的数。Python 的整数**无限大**——只要内存够，写多大的数字都行。

```python
age = 18
temperature = -5
big = 99999999999999999999999999      # 随便多大

# 大数字可以用下划线分隔，方便阅读（Python 3.6+）
world_population = 8_000_000_000      # 80亿
```

### 浮点数 float

有小数点的数。在 Python 里叫"浮点数"。

```python
pi = 3.14159
weight = 65.5
price = -9.9
```

**100 和 100.0 不一样！**

```python
type(100)    # <class 'int'>   → 整数
type(100.0)  # <class 'float'> → 浮点数
```

虽然值看起来一样，但在 Python 心里它们是不同的类型。

### 基本数学运算

```python
3 + 2      # 5   — 加法
10 - 4     # 6   — 减法
6 * 7      # 42  — 乘法

# 除法分两种：
7 / 2      # 3.5 — 普通除法（永远返回浮点数）
7 // 2     # 3   — 整除（只保留整数部分，向下取整）
7 % 2      # 1   — 取余（除不尽剩下的部分）
```

生活中的例子：7 个苹果分给 2 个人，每人 `7 // 2 = 3` 个，剩下 `7 % 2 = 1` 个。

看一个实际用法——把秒数转换成时分秒：

```python
total_seconds = 3725

hours = total_seconds // 3600           # 1 小时
minutes = (total_seconds % 3600) // 60  # 2 分钟
seconds = total_seconds % 60            # 5 秒

print(f"{total_seconds}秒 = {hours}小时{minutes}分钟{seconds}秒")
# 3725秒 = 1小时2分钟5秒
```

判断奇偶数：

```python
number = 7
if number % 2 == 0:
    print(f"{number} 是偶数")
else:
    print(f"{number} 是奇数")
```

### 运算优先级

和数学课一样：先乘除后加减，括号优先。

```python
2 + 3 * 4      # 14（先算 3×4=12，再 2+12）
(2 + 3) * 4    # 20（括号优先）
```

不确定的时候就**加括号**，永远不会错。

### 浮点数的精度问题（小心！）

```python
0.1 + 0.2
# 你可能期待 0.3，但实际得到 0.30000000000000004
```

这不是 Python 的 bug，而是**所有编程语言共同的问题**。就像十进制写不完 `1/3 = 0.33333...`，计算机用二进制也存储不精确某些小数。

解决方法：用 `round()` 四舍五入。

```python
round(0.1 + 0.2, 2)   # 0.3
```

### 乘方的妙用：开方也能用 `**`

```python
2 ** 3       # 8 — 2 的 3 次方
4 ** 0.5     # 2.0 — 平方根（√4）
27 ** (1/3)  # 3.0 — 立方根
# 任何数的 1/n 次方就是它的 n 次方根
```

### 常用的数学函数

```python
abs(-42)             # 42 — 绝对值
max(3, 7, 1)         # 7  — 最大值
min(3, 7, 1)         # 1  — 最小值
round(3.14159, 2)    # 3.14 — 四舍五入保留两位
pow(2, 10)           # 1024 — 相当于 2**10
```

## 六、布尔类型 bool

布尔类型是最简单的类型，只有两个值：

- `True` = 真 = 对 = 是 = 成立
- `False` = 假 = 错 = 否 = 不成立

生活中到处都是布尔值：
- "今天是周末吗？" → True 或 False
- "18 大于等于 18 吗？" → True

### 比较运算产生布尔值

```python
3 > 2            # True（3 确实比 2 大）
3 < 2            # False
5 == 5           # True（注意：两个等号才是比较！）
5 != 3           # True（不等于）
10 >= 10         # True（大于等于）
```

**`=` 和 `==` 完全不同！** `=` 是赋值（放东西进盒子），`==` 是比较（判断左右是否相等）。千万不要搞混。

### 布尔值本质上是数字

```python
True + True    # 2（True 就是 1）
True + False   # 1
True * 10      # 10
False * 10     # 0
```

统计有多少个 True 时很好用：

```python
scores = [True, True, False, True, False]

# 统计 True 的个数
sum(scores)                      # 3

# 统计 False 的个数
len(scores) - sum(scores)        # 2

# True 的比例
sum(scores) / len(scores)        # 0.6（60%）
```

### 什么算 True？什么算 False？

记住这些会变成 **False**：

- `False` 本身
- `0` 和 `0.0`
- `""` 空字符串
- `[]` 空列表
- `{}` 空字典
- `set()` 空集合
- `None`

**其他一切**都是 True，包括负数、非空字符串、非空列表……

```python
bool(0)        # False
bool("")       # False
bool([])       # False
bool(None)     # False

bool(1)        # True
bool(-99)      # True（负数也是 True！）
bool("你好")   # True
bool([1, 2])   # True
```

**记忆口诀：零、空、None 为假，其余为真。**

## 七、None —— 表示"空"

`None` 表示"什么都没有"、"不存在"。

生活中的例子：
- 一个新用户还没填手机号 → 手机号 = `None`
- 问卷里某题没答 → 答案 = `None`

`None` 不是 `0`，不是空字符串 `""`，不是 `False`。`None` 就是 `None`，一种独立的类型。

```python
phone = None

# 判断 None 必须用 is，不能用 ==
if phone is None:
    print("手机号未填写")

if phone is not None:
    print("手机号已填写")
```

## 八、类型：查看与转换

### 查看类型

```python
type(42)        # <class 'int'>
type(3.14)      # <class 'float'>
type(True)      # <class 'bool'>
type("你好")    # <class 'str'>
type(None)      # <class 'NoneType'>
```

### 判断类型（推荐用 isinstance）

```python
isinstance(42, int)            # True
isinstance("你好", str)        # True
isinstance(42, (int, float))   # True — 判断是不是多种类型之一
```

### 类型转换

```python
int("123")         # 字符串 → 整数：123
int(3.9)           # 浮点数 → 整数：3（直接砍掉小数，不是四舍五入！）
int(True)          # True → 1
int(False)         # False → 0

# ⚠️ 这些会报错：
# int("3.14")      # ❌ 字符串里有小数点，不能直接转 int
# int("hello")     # ❌ 这个字符串不是数字

# 正确做法：先转 float 再转 int
int(float("3.14"))  # 3

float("3.14")      # 字符串 → 浮点数：3.14
float(42)          # 整数 → 浮点数：42.0

str(123)           # 数字 → 字符串："123"
str(True)          # 布尔 → 字符串："True"

bool(1)            # True
bool("")           # False
```

类型转换的关键应用——用户输入：

```python
num1_str = "15"     # 用户从键盘输入，永远是字符串
num2_str = "20"

# ❌ 不转换的话，+ 会拼接字符串
print(num1_str + num2_str)   # "1520"

# ✅ 转换后才能做数学运算
num1 = int(num1_str)
num2 = int(num2_str)
print(num1 + num2)            # 35
```

## 九、input —— 和用户对话

`print` 是输出（电脑告诉你），`input` 是输入（你告诉电脑）。

```python
name = input("请输入你的名字：")
print(f"你好，{name}！")

# ⚠️ input 返回的永远是字符串！
age_str = input("请输入年龄：")
age = int(age_str)    # 想做数学运算必须转换类型
print(f"明年你 {age + 1} 岁了")
```

### 小计算器实战

```python
num1 = float(input("第一个数字："))
num2 = float(input("第二个数字："))

print(f"{num1} + {num2} = {num1 + num2}")
print(f"{num1} - {num2} = {num1 - num2}")
print(f"{num1} × {num2} = {num1 * num2}")

if num2 != 0:
    print(f"{num1} ÷ {num2} = {num1 / num2:.2f}")
else:
    print("除数不能为 0")
```

## 十、新手最容易踩的坑

### 坑1：把 `=` 当成"等于"

```python
score = 100        # = 是赋值，不是"等于"

# ❌ 错误写法
if score = 100:    # 这里应该用 == 比较，不是 =
    print("满分")

# ✅ 正确
if score == 100:
    print("满分")
```

**一句话：一个等号是"放进去"，两个等号是"等于吗？"**

### 坑2：字符串 + 数字直接拼接

```python
age = 18
# print("我今年" + age + "岁")   # ❌ 报错！字符串不能和数字用 +

# ✅ 方法一：转成字符串
print("我今年" + str(age) + "岁")

# ✅ 方法二：用 f-string（最推荐）
print(f"我今年{age}岁")

# ✅ 方法三：用逗号分隔
print("我今年", age, "岁")
```

### 坑3：input() 返回的永远是字符串

```python
num = input("输入一个数字：")    # 用户输入 5
print(num * 2)                  # 输出 "55"！因为 "5"*2 = "55"

# ✅ 必须转换类型
num = int(input("输入一个数字："))
print(num * 2)                  # 输出 10
```

### 坑4：int() 不是四舍五入

```python
int(3.9)    # 3 — 直接砍掉小数部分
int(-3.9)   # -3 — 直接砍掉，不是 -4

# ✅ 如果要四舍五入
round(3.9)  # 4
```

### 坑5：0.1 + 0.2 不等于 0.3

```python
0.1 + 0.2 == 0.3   # False！

# 这是因为计算机用二进制存小数，有些十进制的"整"数在二进制是无限循环的。
# 就像你写不完 1/3 = 0.33333... 一样。

# ✅ 比较浮点数用近似值
abs((0.1 + 0.2) - 0.3) < 0.0000001   # True

# ✅ 或用 round
round(0.1 + 0.2, 1) == 0.3           # True
```

### 坑6：变量名用了 Python 关键字

```python
# ❌ 这些都不行
# if = 10          # if 是关键字
# class = "高三"    # class 是关键字
# True = 1          # True 是关键字

# ✅ 加个下划线或者换个名字
class_name = "高三"
is_true = True

# 不确定一个词是不是关键字？
import keyword
print(keyword.kwlist)
```

### 坑7：忘记 print 的括号

```python
# ❌ Python 3 中 print 是函数，必须有括号
# print "Hello"

# ✅
print("Hello")
```
