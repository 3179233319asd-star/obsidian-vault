# 字符串

字符串就是**一串文字**。只要用引号括起来的东西，在 Python 眼里就是字符串——不管里面是汉字、英文还是数字。

⚠️ **"123" 和 123 完全不同！**

```python
"123" + "456"   # "123456" — 文字拼接
123 + 456       # 579 — 数学加法
```

---

## 一、创建字符串

### 三种引号

```python
s1 = '你好'            # 单引号
s2 = "你好"            # 双引号（效果一样）
s3 = """第一行
第二行
第三行"""              # 三引号（可以跨多行，保留换行）
```

单引号和双引号为什么都要有？——当文字里面有引号时，外面用另一种：

```python
quote1 = "他说：'你好'"       # 文字里有单引号，外面用双引号
quote2 = '他说："你好"'       # 文字里有双引号，外面用单引号
quote3 = "他说：\"你好\""     # 或者用反斜杠转义（\ 的意思是"后面的引号不要当代码"）
```

### r"" 原始字符串

反斜杠 `\` 在字符串里有特殊含义（如 `\n` 是换行）。如果你想写一个 Windows 路径，不想让 `\` 被当成转义：

```python
print("C:\\Users\\Desktop")     # 每个 \ 都要写两个，烦
print(r"C:\Users\Desktop")      # r"" 原始字符串，\ 就是 \，不需要转义
```

### str() 转换

```python
str(42)       # "42"
str(3.14)     # "3.14"
str(True)     # "True"
```

## 二、索引——取字符串里的单个字符

字符串里的每个字符都有一个**编号**（索引）。**从 0 开始**（这是计算机的传统，记住就好）。

```
字符串 "Python"：
  字符： P   y   t   h   o   n
  正向： 0   1   2   3   4   5
  反向：-6  -5  -4  -3  -2  -1
```

```python
word = "Python"

word[0]      # "P" — 第1个字符
word[1]      # "y" — 第2个字符
word[-1]     # "n" — 最后一个（负数从右往左数）
word[-2]     # "o" — 倒数第二个

len(word)    # 6 — 字符串长度
```

## 三、切片——取字符串里的一段

语法：`字符串[起始位置:结束位置:步长]`

**关键规则：包含起始位置，不包含结束位置。**（左闭右开）

```
"Hello Python"
 0123456789...

 [0:5]  → "Hello" （0到5，包含0不包含5）
 [6:12] → "Python"（6到12）
```

```python
text = "Hello Python"

text[0:5]     # "Hello"
text[6:12]    # "Python"
text[:5]      # "Hello" — 冒号前面省略 = 从 0 开始
text[6:]      # "Python" — 冒号后面省略 = 到末尾
text[:]       # "Hello Python" — 两边都省略 = 全部

# 带步长
nums = "0123456789"
nums[::2]     # "02468" — 从 0 开始，每 2 个取一个
nums[::3]     # "0369"  — 每 3 个取一个
nums[1::2]    # "13579" — 从索引 1 开始，隔一个取
nums[::-1]    # "9876543210" — 步长为负 = 反转！
```

### 切片不会越界

索引越界会报错，但切片不会——超出部分自动忽略。

```python
"hello"[5:100]   # "o" — 超出部分自动不要了
"hello"[-100:3]  # "hel"
```

## 四、字符串的基本操作

### 拼接 `+`

```python
"Hello" + " " + "World"   # "Hello World"
first = "小"
last = "明"
full = first + last        # "小明"
```

### 重复 `*`

```python
"嘿" * 3    # "嘿嘿嘿"
"-" * 30    # "------------------------------"（分隔线就是这么来的）
```

### 字符串不能和数字拼接！

```python
# "年龄是" + 18      # ❌ 报错！类型不一样不能拼
"年龄是" + str(18)   # ✅ 先把数字转成字符串再拼
```

### 判断包含

```python
"Py" in "Python"     # True
"Java" in "Python"   # False
```

## 五、f-string —— 最推荐的格式化方式

在引号前面加 `f`，然后用 `{}` 包裹变量或表达式。

```python
name = "小明"
age = 18

print(f"我叫{name}，今年{age}岁")
print(f"明年我就{age + 1}岁了")      # {} 里可以做运算

# 控制小数位数
pi = 3.1415926
print(f"π ≈ {pi:.2f}")              # 3.14（保留两位小数）
print(f"π ≈ {pi:.4f}")              # 3.1416

# 对齐
print(f"{'左对齐':<10}|")            # "左对齐      |"
print(f"{'右对齐':>10}|")            # "      右对齐|"
print(f"{'居中':^10}|")              # "    居中    |"

# 数字格式化
big = 1234567890
print(f"{big:,}")                    # 1,234,567,890（自动加千分位逗号）
print(f"{0.8567:.1%}")              # 85.7%（百分比）
print(f"{255:#x}")                   # 0xff（十六进制格式）
```

### 老式格式化（了解即可，不推荐用了）

```python
"我叫%s，今年%d岁" % ("小明", 18)          # % 格式化——Python 2 时代的写法
"我叫{}，今年{}岁".format("小明", 18)       # .format()——Python 3 早期的写法
# 现在都推荐用 f-string，最简单
```

## 七、字符串常用方法

方法是字符串自带的"技能"。格式是 `字符串.方法名()`。**所有方法都返回新字符串，不修改原字符串**。

### 大小写

```python
msg = "hello WORLD"

msg.upper()          # "HELLO WORLD" — 全部大写
msg.lower()          # "hello world" — 全部小写
msg.title()          # "Hello World" — 每个单词首字母大写
msg.capitalize()     # "Hello world" — 仅首字母大写
msg.swapcase()       # "HELLO world" — 大小写互换
```

### 查找

```python
sentence = "我爱学Python，Python很有趣"

sentence.find("Python")       # 3 — 第一次出现的位置
sentence.rfind("o")           # 从右边开始找
sentence.find("Java")         # -1 — 找不到就返回 -1
sentence.index("Python")      # 3 — 和 find 一样，但找不到会报错
sentence.count("Python")      # 2 — 出现了几次
```

`find` 和 `index` 的区别：找不到时 `find` 返回 `-1`，`index` 直接报错。

### 判断（返回 True 或 False）

```python
"123".isdigit()            # True — 全是数字？
"abc".isalpha()            # True — 全是字母？
"abc123".isalnum()         # True — 只有字母和数字？
"  ".isspace()             # True — 全是空白？
"hello".startswith("he")   # True — 以 "he" 开头？
"hello".endswith("lo")     # True — 以 "lo" 结尾？
```

### 去空白

```python
dirty = "   你好世界   "

dirty.strip()     # "你好世界" — 去两边空白
dirty.lstrip()    # "你好世界   " — 去左边空白
dirty.rstrip()    # "   你好世界" — 去右边空白
```

### 分割与合并

```python
data = "苹果,香蕉,橘子,葡萄"

fruits = data.split(",")       # 按逗号分割，得到列表
print(fruits)                  # ["苹果", "香蕉", "橘子", "葡萄"]

# join：用连接符把列表拼回字符串
" → ".join(fruits)             # "苹果 → 香蕉 → 橘子 → 葡萄"
",".join(["a", "b", "c"])      # "a,b,c"
```

`split` 和 `join` 是一对逆向操作。

### 替换

```python
text = "Hello World"
text.replace("World", "Python")   # "Hello Python"
# 注意：原字符串不变，返回新字符串
```

## 八、字符串是不可变的（重要！）

每"修改"一次字符串，实际上是**创建了一个新字符串**，原来的没变。

```python
original = "hello"
new = original.upper()

print(original)   # "hello" — 原字符串还在！
print(new)        # "HELLO" — 新字符串
```

如果要拼接大量字符串（比如几千次循环），用 `join()` 比用 `+` 快很多：

```python
# ❌ 每次 + 都创建新字符串，慢
result = ""
for s in big_list:
    result += s

# ✅ 一次性搞定，快
result = "".join(big_list)
```

## 九、新手最容易踩的坑

### 坑1：以为字符串能原地修改

```python
s = "hello"
# s[0] = "H"        # ❌ 报错！字符串不可变

# ✅ 只能创建新字符串
s = "H" + s[1:]     # "Hello"
```

**任何"修改"字符串的操作，都是创建新字符串返回，原字符串不变。**

### 坑2：忘记 f-string 前面加 f

```python
name = "小明"
print("{name}")         # 输出 {name} — 没加 f，当成普通字符串了
print(f"{name}")        # 输出 小明 — 加 f 才有效
```

### 坑3：find 找不到返回 -1，index 找不到报错

```python
s = "hello"

pos = s.find("z")      # -1 — 不报错
if pos != -1:
    print("找到了")

# pos = s.index("z")   # ❌ ValueError — 直接报错
# 不确定键在不在的时候用 find
```

### 坑4：split 返回列表，不能直接当字符串用

```python
data = "苹果,香蕉,橘子"
parts = data.split(",")     # ["苹果", "香蕉", "橘子"]

# ❌ 以为 split 后可以直接当字符串
print("分隔结果：" + parts)   # 报错！列表和字符串不能拼

# ✅ 遍历或用 join
for item in parts:
    print(item)
print("、".join(parts))
```

### 坑5：循环拼接字符串用 `+` 不用 `join`

```python
# ❌ 慢：每次 + 都在创建新字符串
words = ["a", "b", "c"] * 10000
result = ""
for w in words:
    result += w

# ✅ 快：一次性创建
result = "".join(words)
```

### 坑6：转义字符看不懂

```python
# \n = 换行，\t = Tab，\\ = 反斜杠本身
print("第一行\n第二行")    # 两行
print("路径：C:\\Users")   # 路径：C:\Users

# 如果不想让 \ 被转义，前面加 r
print(r"C:\Users\Desktop")  # 原始字符串，\ 就是 \
```

### 坑7：索引越界

```python
s = "hello"
s[0]      # "h"
s[4]      # "o"
# s[5]    # ❌ IndexError！最后一个索引是 4

# ✅ 切片不会越界（会优雅处理）
s[0:100]  # "hello" — 超出部分自动忽略
```
