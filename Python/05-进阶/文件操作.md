# 文件操作

读写文件，数据持久化。注意配合 [[异常处理]] 处理文件不存在的场景。

## 读文件

```python
# 推荐方式：with 自动关闭
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()      # 全读
    # line = f.readline()   # 读一行
    # lines = f.readlines() # 全读，返回列表
```

## 写文件

```python
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello\n")
    f.writelines(["第一行\n", "第二行\n"])
```

## 文件模式

| 模式 | 含义 |
|------|------|
| `r` | 读取（文件必须存在） |
| `w` | 写入（覆盖） |
| `a` | 追加（不覆盖） |
| `r+` | 读写 |
| `b` | 二进制（如图片） |
