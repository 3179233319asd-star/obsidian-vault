# 异常处理

程序出错不崩溃，优雅处理。

## 基本语法

```python
try:
    num = int(input("输入数字："))
    result = 10 / num
except ValueError:
    print("那不是数字！")
except ZeroDivisionError:
    print("不能除以零！")
except Exception as e:
    print(f"其他错误：{e}")
else:
    print(f"结果是 {result}")   # 没异常才执行
finally:
    print("不管有没有异常都会执行")
```

## 常见异常

| 异常 | 触发场景 |
|------|---------|
| `ValueError` | 类型转换失败 |
| `TypeError` | 操作类型不对 |
| `IndexError` | [[列表与元组\|列表]]索引越界 |
| `KeyError` | [[字典与集合\|字典]]键不存在 |
| `FileNotFoundError` | [[文件操作\|文件]]不存在 |
| `ZeroDivisionError` | 除以零 |

## 自定义异常

```python
class MyError(Exception):
    pass

raise MyError("自定义错误信息")
```
