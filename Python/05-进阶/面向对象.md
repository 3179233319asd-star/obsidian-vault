# 面向对象

[[函数]] 封装逻辑，类封装**数据 + 行为**。

## 定义类

```python
class Dog:
    # 类属性
    species = "犬科"

    # 构造方法
    def __init__(self, name, age):
        self.name = name   # 实例属性
        self.age = age

    # 实例方法
    def bark(self):
        print(f"{self.name} 在叫！")
```

## 继承

```python
class Husky(Dog):
    def bark(self):          # 重写父类方法
        print(f"{self.name} 在嗷呜！")
```

## 特殊方法

```python
class Point:
    def __init__(self, x, y):
        self.x, self.y = x, y

    def __str__(self):
        return f"Point({self.x}, {self.y})"

    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)
```
