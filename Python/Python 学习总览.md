## 核心概念速查

### Python 的哲学

- **用缩进代替花括号** — 缩进在 Python 里是语法，不是风格
- **动态类型** — 变量不需要声明类型，但不能在运算时类型混用
- **一切皆对象** — int、str、list、函数、类都是对象
- **简洁优雅** — "做一件事应该只有一种最好的方式"

### 和 Java/JS 的关键差异

| 特性 | Python | Java | JavaScript |
|------|--------|------|------------|
| 代码块 | 缩进 | `{}` | `{}` |
| 逻辑运算符 | `and` `or` `not` | `&&` `||` `!` | `&&` `||` `!` |
| 整除 | `//` | `/`（整数除整数） | `Math.floor(a/b)` |
| 除法 | `/` 永远返回浮点数 | `/` 取决于类型 | `/` 永远返回浮点数 |
| 自增 | `i += 1` | `i++` | `i++` |
| 空值 | `None` | `null` | `null` / `undefined` |
| 多分支 | `elif` | `else if` | `else if` |
| 比较 | 支持 `a < x < b` | 必须 `x>a && x<b` | 必须 `x>a && x<b` |
| 三元 | `A if 条件 else B` | `条件 ? A : B` | `条件 ? A : B` |
| for 循环 | `for item in list:` | `for (int i=0;...)` | `for (let item of arr)` |
| 多返回值 | 原生支持 `return a,b` | 需要封装对象/数组 | 用数组/对象模拟 |
| 函数参数 | 位置+关键字+默认+可变 | 只有位置参数 | 位置+默认+rest |
| 块级作用域 | ❌ 没有 | ✅ `{}` 有 | ✅ `let/const` 有 |
